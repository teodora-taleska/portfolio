# Teodora Taleska Portfolio

A personal portfolio website showcasing my projects, certificates, blogs, and contact form. Built using **React**, **TailwindCSS**, and **Framer Motion**, this portfolio highlights my work in **Data Science, AI, and full-stack development**.

## 🚀 Features

* **Hero Section:** Introduction with a short personal statement.
* **Projects Section:** Interactive display of key projects with modern styling.
* **Certificates Section:** Carousel of certificates with keyboard and trackpad navigation, smooth animations, and responsive sizing.
* **Experience Section:** Structured display of work and education history.
* **Blogs Section:** Highlights latest posts with clickable links to detailed blog pages.
* **Contact Form:** Secure contact form to send emails without exposing email addresses.
* **Footer:** Social links (LinkedIn, GitHub, Blog) with icons and hover animations.
* **Fully Responsive:** Works on desktop, tablet, and mobile devices.
* **Interactive Animations:** Powered by Framer Motion for smooth transitions.

## 💻 Tech Stack

* **Frontend:** React, TailwindCSS
* **Animations:** Framer Motion
* **Email Integration:** [EmailJS](https://www.emailjs.com/)
* **Icons:** Inline SVG for lightweight social links

## ✉️ Contact Form Setup

This project uses **EmailJS** to send emails without exposing your personal address.

1. Create an account at [EmailJS](https://www.emailjs.com/).
2. Create a service and template for your email.
3. Add your **Service ID**, **Template ID**, and **User ID** in the `ContactForm.jsx` file.
4. Test by sending a message through the form.
